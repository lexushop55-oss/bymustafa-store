# MUSTAFA STORE AI SYSTEM

Всегда анализируй задачу перед началом работы.

Используй подходящий skill из папки skills.

Если задача связана:
- с UI → используй ui-ux-pro-max
- с frontend → используй frontend-design
- с конверсией → используй conversion-ui
- с текстами → используй copywriting
- с анализом дизайна → используй design-review

Всегда:
- сохраняй clean modern aesthetic
- избегай дешевого визуала
- улучшай spacing
- улучшай typography
- делай premium UI
- анализируй UX
- проверяй mobile responsiveness
- улучшай visual hierarchy
- сохраняй consistent design system

Перед изменением:
1. анализируй текущую структуру
2. находи слабые места
3. предлагай улучшения
4. только потом изменяй код

Никогда:
- не ломай структуру проекта
- не создавай хаос в CSS
- не делай cluttered UI
- не используй outdated design patterns

Всегда думай как senior product designer + senior frontend architect.
